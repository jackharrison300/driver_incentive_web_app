# Architecture Decisions

- Fully serverless
- "Infrastructure as code"
- Full-stack web framework Remix
- AWS Cognito Auth
    - No stored session data
- Prisma ORM
- MySQL back-end
- React.js front-end
    - Tailwind CSS
    - Headless UI Component Library
